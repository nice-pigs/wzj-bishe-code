## 🎯 检测算法

```
压力传感器（任一通道有压力）：70分 ← 主要依据
超声波测距（<50cm）：25分
PIR传感器（检测到运动）：5分

总分 >= 70分 → 判定为占用
总分 < 70分 → 判定为空闲
```

## 💡 为什么选择PB4和PB5？

1. ✅ **完全空闲** - 没有被任何模块占用
2. ✅ **容易找到** - 在开发板右侧，连续的两个引脚
3. ✅ **同一组** - 都在GPIOB，代码简洁
4. ✅ **位置好** - 在PB0/PB1（超声波）附近，便于接线

---

## ⚠️ 重要提示

1. **压力传感器有3个GND**，都要接到STM32的GND
2. **超声波需要5V供电**，不是3.3V
3. **所有设备必须共地**，否则信号不稳定
4. **LED必须串联220Ω电阻**，否则会烧坏
