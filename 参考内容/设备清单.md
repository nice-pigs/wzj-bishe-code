# 座位占用检测系统 - 设备清单

## 一、核心控制模块

### 1. 主控单片机
- **型号**：STM32F103C8T6
- **数量**：1个
- **功能**：系统核心控制器，负责传感器数据采集、逻辑判断、设备控制
- **主要参数**：
  - 内核：ARM Cortex-M3
  - 主频：72MHz
  - Flash：64KB
  - RAM：20KB
  - GPIO：37个
  - ADC：2个12位ADC
  - 定时器：7个
  - 串口：3个USART

### 2. WiFi通信模块
- **型号**：ESP32-S3
- **数量**：1个
- **功能**：WiFi无线通信，连接云平台，实现远程数据上报和控制
- **主要参数**：
  - 内核：Xtensa 32位 LX7双核处理器
  - 主频：240MHz
  - WiFi：802.11 b/g/n
  - 蓝牙：Bluetooth 5 (LE)
  - Flash：可选4MB/8MB/16MB
  - RAM：512KB SRAM
  - GPIO：45个可编程GPIO
- **接口**：串口通信（UART）与STM32连接

## 二、传感器模块（3个）

### 1. DHT11 温湿度传感器
- **型号**：DHT11
- **数量**：1个
- **功能**：检测环境温度和湿度，辅助判断座位使用环境
- **主要参数**：
  - 温度范围：0-50°C，精度±2°C
  - 湿度范围：20-90%RH，精度±5%RH
  - 通信方式：单总线数字信号
  - 供电电压：3.3V-5V
  - 采样周期：≥1秒
- **引脚定义**：
  - VCC：电源正极（3.3V）
  - DATA：数据线（连接STM32 PA0）
  - GND：电源负极

### 2. 光敏电阻模块
- **型号**：光敏电阻传感器模块
- **数量**：1个
- **功能**：检测环境光照强度，辅助判断座位区域是否有人（遮挡光线）
- **主要参数**：
  - 检测范围：0-1023（ADC值）
  - 输出方式：模拟输出（AO）+ 数字输出（DO）
  - 供电电压：3.3V-5V
  - 灵敏度：可通过电位器调节
- **引脚定义**：
  - VCC：电源正极（3.3V）
  - GND：电源负极
  - AO：模拟输出（连接STM32 PA1，ADC输入）
  - DO：数字输出（连接STM32 PA2，可选）

### 3. HC-SR501 人体红外传感器
- **型号**：HC-SR501 PIR人体红外传感器
- **数量**：1个
- **功能**：检测座位区域是否有人体活动，核心传感器
- **主要参数**：
  - 检测距离：3-7米（可调）
  - 检测角度：<120度
  - 延迟时间：5-200秒（可调）
  - 触发方式：L不可重复触发 / H可重复触发（跳线选择）
  - 输出电平：高电平3.3V（检测到人体）
  - 供电电压：5V（也可3.3V）
  - 静态电流：<50uA
- **引脚定义**：
  - VCC：电源正极（5V或3.3V）
  - OUT：信号输出（连接STM32 PB8）
  - GND：电源负极
- **调节旋钮**：
  - Sx：灵敏度调节（检测距离）
  - Tx：延迟时间调节

## 三、执行器模块（4个）

### 1. OLED显示屏（带4按键）
- **型号**：0.96寸OLED显示屏（SSD1306驱动芯片）
- **数量**：1个
- **功能**：本地显示座位状态、传感器数据、系统信息
- **主要参数**：
  - 尺寸：0.96英寸
  - 分辨率：128×64像素
  - 驱动芯片：SSD1306
  - 通信方式：I2C
  - I2C地址：0x78（7位地址0x3C）
  - 供电电压：3.3V-5V
  - 显示颜色：单色（蓝色或白色）
- **引脚定义**：
  - GND：电源负极
  - VCC：电源正极（3.3V）
  - SCL：I2C时钟线（连接STM32 PB6）
  - SDA：I2C数据线（连接STM32 PB7）
  - K4：按键4（连接STM32 PB15）
  - K3：按键3（连接STM32 PB14）
  - K2：按键2（连接STM32 PB13）
  - K1：按键1（连接STM32 PB12）
- **按键功能**（可编程）：
  - K1：手动刷新传感器数据
  - K2：切换显示模式
  - K3：手动设置座位为"占用"
  - K4：手动设置座位为"空闲"

### 2. 蜂鸣器模块
- **型号**：有源蜂鸣器模块（低电平触发）
- **数量**：1个
- **功能**：声音提示，座位状态变化时发出提示音
- **主要参数**：
  - 类型：有源蜂鸣器
  - 触发方式：低电平触发（IO口输出低电平时响）
  - 工作电压：3.3V-5V
  - 工作电流：<30mA
  - 频率：约2700Hz
- **引脚定义**：
  - VCC：电源正极（5V）
  - GND：电源负极
  - I/O：信号输入（连接STM32 PA4）
- **控制逻辑**：
  - PA4输出低电平（0V）→ 蜂鸣器响
  - PA4输出高电平（3.3V）→ 蜂鸣器不响

### 3. 指示灯1（绿色LED）
- **型号**：LED发光二极管（绿色）
- **数量**：1个
- **功能**：指示座位"空闲"状态
- **主要参数**：
  - 颜色：绿色
  - 工作电压：2.0-2.2V
  - 工作电流：10-20mA
  - 正向压降：约2.0V
- **连接方式**：
  - STM32 PA5 → 220Ω限流电阻 → LED正极 → LED负极 → GND
- **控制逻辑**：
  - PA5输出高电平（3.3V）→ LED亮（座位空闲）
  - PA5输出低电平（0V）→ LED灭

### 4. 指示灯2（红色LED）
- **型号**：LED发光二极管（红色）
- **数量**：1个
- **功能**：指示座位"占用"状态
- **主要参数**：
  - 颜色：红色
  - 工作电压：1.8-2.0V
  - 工作电流：10-20mA
  - 正向压降：约1.8V
- **连接方式**：
  - STM32 PA6 → 220Ω限流电阻 → LED正极 → LED负极 → GND
- **控制逻辑**：
  - PA6输出高电平（3.3V）→ LED亮（座位占用）
  - PA6输出低电平（0V）→ LED灭

## 九、注意事项

1. **电平匹配**：
   - STM32 GPIO输出电平：3.3V
   - 蜂鸣器模块：低电平触发（注意反逻辑）
   - LED：需串联220Ω限流电阻

2. **供电要求**：
   - STM32：3.3V（板载稳压）
   - ESP32-S3：5V（USB供电或外部5V）
   - 传感器：大部分支持3.3V-5V
   - 建议使用5V 2A电源适配器或移动电源

3. **通信波特率**：
   - STM32调试串口（PA9/PA10）：115200
   - STM32与ESP32通信（PB10/PB11）：115200

4. **I2C地址**：
   - OLED：0x78（7位地址0x3C）

5. **DHT11使用**：
   - 上电后需等待2秒稳定
   - 读取间隔≥1秒

6. **PIR传感器**：
   - 上电后需预热1分钟
   - 调节灵敏度和延迟时间以适应座位检测
