# 硬件终端设计与制作指南

## 1. 系统架构概述

本硬件终端采用 **STM32单片机** 作为核心控制器，通过传感器检测座位使用状态，并利用 Wi-Fi 模块将数据上传至云端服务器。

### 系统组成
```
传感器模块 → STM32单片机 → Wi-Fi模块 → 云端服务器
                ↓
           本地显示模块
```

## 3. 电路设计

### 3.1 最小系统电路

STM32F103C8T6 最小系统包括：
- 电源电路（3.3V稳压）
- 复位电路
- 时钟电路（外部8MHz晶振）
- 启动配置电路

```
                    STM32F103C8T6
                  ┌─────────────────┐
        VCC 3.3V ─┤VDD         VSS ├─ GND
                  │                 │
        Reset ───┤NRST        PA13 │── SWDIO
                  │                 │
        8MHz ────┤OSC_IN      PA14 │── SWCLK
                  │                 │
                  │     ...         │
                  └─────────────────┘
```

### 3.2 传感器接口电路

**压力传感器接口（模拟量）：**
```
压力传感器 ────┬─── PA0 (ADC1_IN0)
              │
              ├─── 10KΩ ──── VCC
              │
              └─── 100nF ──── GND (滤波)
```

**红外传感器接口（数字量）：**
```
HC-SR501 OUT ──── PA1 (GPIO_Input)
               │
               └─── 10KΩ ──── GND (下拉)
```

### 3.3 Wi-Fi 模块接口电路

```
STM32           ESP8266
PA9 (TX)  ────  RX
PA10(RX)  ────  TX
PB0 (GPIO) ──── RST (复位)
3.3V      ────  VCC
GND       ────  GND
```

**注意事项：**
- ESP8266需要独立的3.3V稳压电源（峰值电流可达300mA）
- 添加100μF电容滤波，防止电源波动

### 3.4 显示模块接口电路

**I2C接口：**
```
STM32           OLED (SSD1306)
PB6 (I2C1_SCL) ──── SCL
PB7 (I2C1_SDA) ──── SDA
3.3V           ──── VCC
GND            ──── GND
```

### 3.5 指示灯电路

```
红色LED (占用状态):
PB1 (GPIO) ──── 1KΩ ──── LED(+) ──── LED(-) ──── GND

绿色LED (空闲状态):
PB2 (GPIO) ──── 1KΩ ──── LED(+) ──── LED(-) ──── GND
```

## 4. PCB 设计建议

### 4.1 布局原则
1. **电源模块**放置在PCB边缘，便于连接
2. **STM32芯片**居中放置，周围布置外围电路
3. **Wi-Fi模块**远离晶体振荡器，避免干扰
4. **传感器接口**放置在PCB边缘，便于连接外部传感器

### 4.2 布线原则
1. **电源线**加宽（≥20mil），降低压降
2. **地线**形成环路，提高抗干扰能力
3. **高频信号线**（如晶振）尽量短，避免干扰
4. **模拟信号**与**数字信号**分开走线

### 4.3 PCB 尺寸建议
- 建议尺寸：50mm × 70mm
- 板厚：1.6mm
- 层数：2层板（经济型）

## 9. 常见问题

### Q1: 为什么选择STM32而不是Arduino？
**A:** STM32性能更强、成本更低、更适合批量生产。Arduino适合快速原型开发，但在实际产品中STM32更具优势。

### Q2: 压力传感器和红外传感器如何选择？
**A:** 
- **压力传感器**：检测更准确，适合固定座位
- **红外传感器**：成本更低，非接触式，适合公共场所

### Q3: ESP8266能否替换为其他Wi-Fi模块？
**A:** 可以，推荐替换方案：
- **ESP32**：功能更强，支持蓝牙，价格稍高（¥15-25）
- **ESP8285**：ESP8266的升级版，内置Flash（¥12-18）

### Q4: 如何保证Wi-Fi连接稳定性？
**A:** 
1. 确保电源稳定（添加大容量电容）
2. Wi-Fi模块远离干扰源
3. 实现断线重连机制
4. 定期发送心跳包
